# recipes

one folder per walkthrough. each recipe ships with:

- `README.md`: what it does, the hardware it was tested on, prerequisites, the exact commands in order, how to verify it worked, and rough running cost
- the actual config files and scripts, runnable as-is

the bar: every command in a recipe has been run on real hardware before publishing. no copy-paste from docs, no untested steps, no "should work".
